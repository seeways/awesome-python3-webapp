
## 主要架构
```
awesome-python3-webapp/
 +- www/                     <-- web目录
 |   |
 |   +- static/               <-- 静态文件
 |   |
 |   +- templates/            <-- 模板文件
 |   |
 |   +- README.md             <-- 说明文件
 |
 +- log/                     <-- 日志目录
 |
 +- conf/                    <-- 配置目录
```


## 三方库

```
# 异步框架aiohttp
pip install aiohttp

# 前端模板引擎jinja2
pip install jinja2

# MySQL的Python异步驱动程序aiomysql(首先装了mysql)
pip install aiomysql

# 监控
pip install watchdog
```
